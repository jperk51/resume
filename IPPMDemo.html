Image Processing Package Manager (IPPM) was created in NodeJS as a demostration of backend code built to allow for new image processing algorithms to be plug and played similarly to how Node handles libraries. The demo shows the user a page for data input, x, y, array of numbers, then the page has buttons at the bottom that correspond to library calculations to be done on the backend. The demo also has another page allowing the user to select which libraries they want installed to the frontend. Once the libraries are installed the user then can execute the selected calculations via a .dll or .so on the server. (Click images below to enlarge)
