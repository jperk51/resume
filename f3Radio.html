The Falcon III Radio is a tactical radio developed by Harris Corp. The Falcon III supports many different waveforms for communication. I tested the radio based on guided tests; when bugs were found, I tracked, debugged, and fixed them. The Falcon III runs on an embedded Linux-based OS, and most of the code I worked with was DSP, digital signal processor, code written in C or C++. (Click images below to enlarge)
