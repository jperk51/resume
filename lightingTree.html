My first project for the Electronics Lab at OSU was to complete a drag race lighting tree. The lighting tree was already built by a coworker in the lab before I was hired. My task was to finish wiring and program the lighting tree so it would count down to start the way a real drag strip lighting tree does. Once the lighing tree was working, we decided to add pressure pads at the start and finish of the strip so that the program could record exactly how long the race took for each cart. The cart were designed to hold the compressed air engines created by the capstone students. (Click images below to enlarge)
